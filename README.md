# Dicionário

## Objetivo do Projeto
<p>O objetivo do projeto é fazer um site em que seja possível ver tradução de algumas palavras em inglês, junto com pronúncia. Foca principalmente no aprendizado sobre manipulação do DOM e tocar áudios.</p>

## Requisitos do Projeto
<ul>
  <li>Possuir a tradução de palavras;</li>
  <li>Ter um botão para ouvir a pronúncia das palavras;</li>
  <li>Ter um modal, que abre e fecha;</li>
  <li>As palavras virem de um arquivo JSON.</li>
</ul>

## Linguagens utilizadas
<ul>
  <li>HTML;</li>
  <li>CSS;</li>
  <li>JavaScript.</li>
</ul>
